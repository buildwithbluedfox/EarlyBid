# 🚀 EarlyBid

EarlyBid is a professional, free application designed to simplify the way freelancers and small businesses create, manage, and share product or service invoices and estimates.

## 🌟 Overview

EarlyBid removes the friction from billing. Whether you are sending a quick estimate for a new project or a detailed invoice for services rendered, EarlyBid provides the tools to make your professional documents look polished and consistent.

## ✨ Key Features

- **Effortless Invoice Creation**: Generate professional invoices in minutes.
- **Estimate Management**: Send detailed quotes to clients and convert them to invoices seamlessly.
- **Template Library**: Access a growing library of free, professionally designed invoice and estimate templates.
- **Custom HTML Templates**: For power users, EarlyBid supports custom HTML-based templates, giving you complete control over your brand's visual identity.
- **Client Sharing**: Quickly share documents with your clients via secure links or exports.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Runtime/Package Manager**: [Bun](https://bun.sh/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

Ensure you have [Bun](https://bun.sh/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd earlybid
   ```

2. **Install dependencies:**
   ```bash
   bun install
   ```

3. **Run the development server:**
   ```bash
   bun run dev
   ```

4. **Open the app:**
   Navigate to `http://localhost:3000` in your browser.
